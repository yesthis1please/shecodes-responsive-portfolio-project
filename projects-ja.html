Vanilla JS Weather App

Vanilla JS - HTML - CSS - JavaScript - Editor - API- GitHub - Bootstrap - Netlify

This weather app was built with Vanilla JavaScript as the final project for the SheCodes Plus Workshop. It uses the OpenWeatherMap API which gives you real-life data and forecasts for any location around the world. Find your city with the search function or simply use your current location by clicking on the icon. It will show the current temperature, weather condition, wind speed and provide a six-day forecast.
